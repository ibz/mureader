muSocial is a minimalist feed reader. Or an anti-social network. You can use it to follow blogs that publish RSS / Atom feeds, which most blogs do.

Simple design that stays out your way. Inspired by Hacker News and Pinboard.

All about privacy. No tracking. No ads.

Open source. You can browse the source code on GitHub. You can easily run it on your own server.

If you don't want to run it yourself, go ahead and create an account on this instance. It is free for now, but I will probably charge for usage in the future, as I don't plan to monetize it by tracking you or running ads.
